# Build dotplots for RunGseaEnrich results

Build dotplots for RunGseaEnrich results

## Usage

``` r
.build_gsea_enrich_plots(enricher_result, gsea_result)
```

## Arguments

- enricher_result:

  compareCluster enricher result (or NULL).

- gsea_result:

  compareCluster GSEA result.

## Value

A list of plot lists.
