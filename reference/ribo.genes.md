# Ribosomal gene symbols (human)

A character vector of 184 human ribosomal protein gene symbols,
including cytoplasmic (RPL/RPS) and mitochondrial (MRPL/MRPS) ribosomal
genes. Originally from the ProjectSVR package.

## Usage

``` r
ribo.genes
```

## Format

A character vector of length 184.
